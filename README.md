# Repositório criado para os desafios de React.

## Desafio 1: Tela de Comentários

## Objetivo

Criar uma pequena tela de comentários utilizando **React + Vite**.

O objetivo não é desenvolver algo grande, mas praticar bem os fundamentos de Front-End com React, CSS e acessibilidade.

---

## Requisitos do desafio

1. Criar uma aplicação em **React + Vite**.

2. A tela deve ter um título, por exemplo:

   ```txt
   Comentários
   ```

3. Adicionar um campo de texto usando:

   ```html
   <textarea>
   ```

4. Adicionar um botão para incluir o comentário.

5. Exibir em uma lista os comentários que forem adicionados.

6. Criar uma validação simples para não permitir comentário vazio.

7. Fazer um layout simples, mas bem cuidado com CSS.

8. Aplicar boas práticas de acessibilidade, como:

   * usar `<label>` associado ao `<textarea>`;
   * usar um botão com texto claro;
   * garantir foco visível nos campos;
   * usar HTML semântico;
   * exibir mensagens de erro compreensíveis.

---

## Conceitos praticados

Neste desafio, serão praticados os seguintes conceitos:

* setup com Vite;
* componentes React;
* estado com `useState`;
* eventos de formulário;
* renderização de listas;
* CSS organizado;
* acessibilidade desde o começo.

---

## Resultado esperado

Ao final do desafio, a aplicação deve permitir que a pessoa usuária:

* escreva um comentário em um campo de texto;
* clique em um botão para adicionar o comentário;
* veja o comentário aparecer em uma lista;
* receba uma mensagem de erro caso tente adicionar um comentário vazio;
* consiga navegar e interagir com a tela de forma acessível.
